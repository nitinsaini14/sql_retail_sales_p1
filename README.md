## 📌 Project Overview

This project analyzes a retail sales dataset using SQL in PostgreSQL. The objective is to perform data cleaning, exploratory data analysis (EDA), and solve real-world business problems using SQL queries.

The project demonstrates essential SQL skills required for Data Analyst roles, including filtering, aggregation, grouping, date/time analysis, and customer sales analysis.

# 📊 Exploratory Data Analysis (EDA)

The following business questions were solved using SQL.

---

## 1️⃣ Total Sales Transactions

**Question:** Find the total number of sales transactions in the dataset.

```sql
SELECT COUNT(transactions_id) AS total_transactions
FROM retail_sales;
```

---

## 2️⃣ Unique Customers

**Question:** Find the total number of unique customers who made purchases.

```sql
SELECT COUNT(DISTINCT customer_id) AS total_unique_customers
FROM retail_sales;
```

---

## 3️⃣ Unique Product Categories

**Question:** Display all unique product categories.

```sql
SELECT DISTINCT category
FROM retail_sales
ORDER BY category;
```

---

## 4️⃣ Total Sales Revenue

**Question:** Calculate the total sales revenue generated by the store.

```sql
SELECT SUM(total_sale) AS total_sales
FROM retail_sales;
```

---

## 5️⃣ Electronics Transactions

**Question:** Retrieve the first 5 Electronics transactions.

```sql
SELECT *
FROM retail_sales
WHERE category = 'Electronics'
LIMIT 5;
```

---

## 6️⃣ Sales on 2022-11-05

**Question:** Retrieve all sales made on **2022-11-05**.

```sql
SELECT *
FROM retail_sales
WHERE sale_date = '2022-11-05';
```

---

## 7️⃣ Clothing Sales Analysis

**Question:** Find all Clothing transactions where quantity sold is greater than 10 during November 2022.

```sql
SELECT *
FROM retail_sales
WHERE category = 'Clothing'
  AND quantity > 10
  AND sale_date BETWEEN '2022-11-01' AND '2022-11-30';
```

---

## 8️⃣ Category-wise Total Sales

**Question:** Calculate total sales for each category.

```sql
SELECT
    category,
    SUM(total_sale) AS total_sales
FROM retail_sales
GROUP BY category;
```

---

## 9️⃣ Average Age of Beauty Customers

**Question:** Find the average age of customers purchasing Beauty products.

```sql
SELECT AVG(age) AS average_age
FROM retail_sales
WHERE category = 'Beauty';
```

---

## 🔟 High Value Transactions

**Question:** Find all transactions where total sales exceeded **1000**.

```sql
SELECT *
FROM retail_sales
WHERE total_sale > 1000;
```

---

## 1️⃣1️⃣ Gender-wise Transactions

**Question:** Calculate the total number of transactions made by each gender in every category.

```sql
SELECT
    gender,
    category,
    COUNT(transactions_id) AS total_transactions
FROM retail_sales
GROUP BY gender, category;
```

---

## 1️⃣2️⃣ Monthly Sales Analysis

**Question:** Calculate the average sales for each month.

```sql
SELECT
    EXTRACT(YEAR FROM sale_date) AS year,
    EXTRACT(MONTH FROM sale_date) AS month,
    AVG(total_sale) AS average_sales
FROM retail_sales
GROUP BY year, month
ORDER BY year, average_sales DESC;
```

---

## 1️⃣3️⃣ Top 5 Customers

**Question:** Find the top 5 customers based on total sales.

```sql
SELECT
    customer_id,
    SUM(total_sale) AS total_sales
FROM retail_sales
GROUP BY customer_id
ORDER BY total_sales DESC
LIMIT 5;
```

---

## 1️⃣4️⃣ Unique Customers by Category

**Question:** Find the number of unique customers purchasing items from each category.

```sql
SELECT
    category,
    COUNT(DISTINCT customer_id) AS unique_customers
FROM retail_sales
GROUP BY category
ORDER BY unique_customers DESC;
```

---

## 1️⃣5️⃣ Shift-wise Orders

**Question:** Categorize orders into Morning, Afternoon, and Evening shifts and count the number of orders.

```sql
SELECT
    CASE
        WHEN EXTRACT(HOUR FROM sale_time) < 12 THEN 'Morning'
        WHEN EXTRACT(HOUR FROM sale_time) BETWEEN 12 AND 17 THEN 'Afternoon'
        ELSE 'Evening'
    END AS shift,
    COUNT(*) AS number_of_orders
FROM retail_sales
GROUP BY shift
ORDER BY number_of_orders DESC;
```
